Robot arm project:

To be able to control a robot manipulator as required by its operation, it is important to consider the kinematic in the design of the control algorithm. In robotics, the kinematic descriptions of manipulators and their assigned tasks are utilized to set up the fundamental equations for dynamics and control. here we are focusing on knowing the workspace of our robot arm as well as being able to pick and place tasks. And finally we will buit an andoid application using app inventor to control the robot arm.

some information about the "Braccio" used in this project

Link 0 (base), joint 1 (base rot), Link 1 (71.5mm), Joint 2 (Shoulder), Link 2 (125mm), Joint 3 (Elbow), Link 3 (125mm), joint 4 (wrist), Link 4 (60mm), joint 5 (wrist rot), Grip(132mm).

Why Forward kinematics is important for robot arm

Forward kinematics is frequently used to calculate the position of the end effector when we know the degree value of each joint. By giving every joint a degree, we can know where the end of the robot arm is located. the photo bellow show the MATLAB result

Why inverse kinematics is important for robot arm

In robotics, Inverse Kinematics is the most used, the Inverse kinematics are used for picking up objects. When I have a specific object with known coordination, I can generate the angle to let the robot arm go to this position. the photo bellow show the MATLAB result

Arduino Robot Arm with Smartphone Control

Using the sliders in the app we can manually control the movement of each servo or axis of the robot arm. Also using the “Save” button we can record each position or step and then the robot arm can automatically run and repeat these steps. With the same button we can pause the automatic operation as well as reset or delete all steps so that we can record new ones.

Projet bras robotisé :

Pour être en mesure de contrôler un robot manipulateur tel que requis par son fonctionnement, il est important de considérer la cinématique dans la conception de l'algorithme de contrôle. En robotique, les descriptions cinématiques des manipulateurs et leurs tâches assignées sont utilisées pour établir les équations fondamentales de la dynamique et du contrôle. ici, nous nous concentrons sur la connaissance de l'espace de travail de notre bras de robot ainsi que sur la capacité de choisir et de placer des tâches. Et enfin, nous allons construire une application Android utilisant l'inventeur de l'application pour contrôler le bras du robot.

Quelques informations sur le "Braccio" utilisé dans ce projet

Lien 0 (base), articulation 1 (pourriture de la base), lien 1 (71,5 mm), articulation 2 (épaule), lien 2 (125 mm), articulation 3 (coude), lien 3 (125 mm), articulation 4 (poignet), Lien 4 (60 mm), joint 5 (pourriture du poignet), Grip (132 mm).

Pourquoi utiliser la cinématique est importante pour le bras du robot

La cinématique directe est fréquemment utilisée pour calculer la position de l'effecteur lorsque nous connaissons la valeur en degrés de chaque articulation. En donnant un degré à chaque articulation, nous pouvons savoir où se trouve l'extrémité du bras du robot. la photo ci-dessous montre le résultat MATLAB

Pourquoi utiliser la cinématique inverse est importante pour le bras du robot

En robotique, la cinématique inverse est la plus utilisée, la cinématique inverse est utilisée pour ramasser des objets. Lorsque j'ai un objet spécifique avec une coordination connue, je peux générer l'angle pour laisser le bras du robot aller dans cette position. la photo ci-dessous montre le résultat MATLAB

Bras robotique contrôler par le smartphone

En utilisant les curseurs de l'application, nous pouvons contrôler manuellement le mouvement de chaque servo ou axe du bras du robot. En utilisant également le bouton "Enregistrer", nous pouvons enregistrer chaque position ou étape, puis le bras du robot peut automatiquement exécuter et répéter ces étapes. Avec le même bouton, nous pouvons mettre en pause le fonctionnement automatique ainsi que réinitialiser ou supprimer toutes les étapes afin de pouvoir en enregistrer de nouvelles.
