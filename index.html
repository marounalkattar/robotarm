Robot arm project:

To be able to control a robot manipulator as required by its operation, it is important to consider the kinematic in the design of the control algorithm. In robotics, the kinematic descriptions of manipulators and their assigned tasks are utilized to set up the fundamental equations for dynamics and control. here we are focusing on knowing the workspace of our robot arm as well as being able to pick and place tasks. And finally we will buit an andoid application using app inventor to control the robot arm.

some information about the "Braccio" used in this project

Link 0 (base), joint 1 (base rot), Link 1 (71.5mm), Joint 2 (Shoulder), Link 2 (125mm), Joint 3 (Elbow), Link 3 (125mm), joint 4 (wrist), Link 4 (60mm), joint 5 (wrist rot), Grip(132mm).

Why Forward kinematics is important for robot arm

Forward kinematics is frequently used to calculate the position of the end effector when we know the degree value of each joint. By giving every joint a degree, we can know where the end of the robot arm is located.

Why inverse kinematics is important for robot arm

In robotics, Inverse Kinematics is the most used, the Inverse kinematics are used for picking up objects. When I have a specific object with known coordination, I can generate the angle to let the robot arm go to this position.

Arduino Robot Arm with Smartphone Control

Using the sliders in the app we can manually control the movement of each servo or axis of the robot arm. Also using the “Save” button we can record each position or step and then the robot arm can automatically run and repeat these steps. With the same button we can pause the automatic operation as well as reset or delete all steps so that we can record new ones.
